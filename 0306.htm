3.    建立 Ceph Cluster

 

規劃電腦，總共三台實體主機，製作四個VM

三個VM包含OSD、MON服務、另外一台包含MDS服務

 

在每台電腦建立KVM 資料夾，放置Ceph KVM模板

$ mkdir KVM

$ scp 140.120.8.112:$HOME/0303/ceph-kvm-template.img KVM

$ cd KVM

$ wget http://140.120.8.53/0303/Config-Ceph-KVM

$ chmod +x Config-Ceph-KVM

 

重新命名模板、並快速上線

$ mv ceph-kvm-template.img ceph-101.img

$ Config-Ceph-KVM ceph-151.img 140.120.8.151 151

HostName: ceph-151

TapName : tap151

IP      : 140.120.8.151

Gateway : 140.120.8.1

FakeMac : 94:de:80:5d:a1:08

SockDir=/home/hsu/0303

ceph-151.img mount on /mnt/tmp

Umount ceph-151.img

 

指定 /dev/sdc 為OSD硬碟

$ ./add-osd start-ceph-151-151-AsDaemon /dev/sdc

Done!

 

VM 上線

$ start-ceph-151-151-AsDaemon
